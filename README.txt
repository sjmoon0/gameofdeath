This is the Game of Death
Project for COSC625
Frostburg State University

An HTML5/javascript multiplayer board game that uses Node.js with socket.io. The objective of the game is to stay alive, but that may prove more difficult than imagined.

Technologies used:
Node.js
Socket.io
Bootstrap
jQuery
